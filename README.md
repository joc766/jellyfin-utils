# Python Package for Jellyfin Utilities
This project defines utilities to orchestrate the `makemkvcon`, `ffmpeg`,
`rsync`, and organizational methods for my jellyfin server.


## Structure

### makemkv_tools
Defines the utilites to identify a drive, get disc, title, and stream info.

To come: `makemkv mkv` command utilities

### cli.py
The entry point for the main CLI.
